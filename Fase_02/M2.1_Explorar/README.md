# M2.1_Explorar: Análisis y Exploración Inicial de Datos

Este directorio contiene cuadernos de Jupyter (.ipynb) dedicados a la fase inicial de exploración y análisis de datos. Aquí encontrarás los pasos y visualizaciones clave para comprender la estructura, las distribuciones y las relaciones dentro de los datos.

## Cuadernos de Jupyter

A continuación, se listan los cuadernos de Jupyter contenidos en esta carpeta:

* **`01_Carga_y_Limpieza_Inicial.ipynb`**: Cuaderno centrado en la carga de los datos, la inspección inicial, el manejo de valores nulos y la preparación básica para el análisis.
* **`02_Analisis_Estadistico_Descriptivo.ipynb`**: Explora las estadísticas descriptivas de las variables, incluyendo medidas de tendencia central, dispersión y forma.
* **`03_Visualizacion_Univariante.ipynb`**: Contiene gráficos y visualizaciones para entender la distribución de variables individuales (histogramas, box plots, etc.).
* **`04_Visualizacion_Bivariante.ipynb`**: Se enfoca en la relación entre pares de variables, utilizando gráficos como diagramas de dispersión, heatmaps y gráficos de barras agrupadas.

## Tus Contribuciones y Gráficos Adicionales

Si has creado versiones adicionales o cuadernos con más gráficos y análisis, te animo a añadirlos aquí y listarlos en esta sección. Por ejemplo:

* **`05_Exploracion_Profundidad_Variables_Categoricas_Tu_Version.ipynb`**: (Ejemplo) Tu cuaderno con análisis detallado de variables categóricas.
* **`06_Relaciones_Multiples_Tu_Version.ipynb`**: (Ejemplo) Tu cuaderno explorando relaciones entre múltiples variables o visualizaciones avanzadas.
